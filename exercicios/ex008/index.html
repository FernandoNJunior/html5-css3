Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado, utiliza-se para isso a tag <del>.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois, podemos utilizar para tanto a tag <ins>.

Ou também, existe para essa função de texto sublinhado a tag<u>, (sem semântica).
